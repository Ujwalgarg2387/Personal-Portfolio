# Ujwal Garg — Developer Portfolio

A personal portfolio website built to showcase my work, skills, and experience as a Full Stack Software Developer.

## 🔗 Live Demo

**https://ujwal-garg-portfolio.netlify.app/**

## 🛠️ Tech Stack

- **React.js** + **Vite**
- **Tailwind CSS**
- **Formspree** — contact form
- **Lucide React** — icons

## ✨ Features

- Dark / Light theme toggle
- Interactive terminal with portfolio commands
- Scroll-reveal animations
- Project cards
- Contact form with email delivery
- Fully responsive

## 🚀 Running Locally

```bash
git clone https://github.com/Ujwalgarg2387/portfolio
cd portfolio
npm install
npm run dev
```

## 📁 Customization

All personal data (name, bio, projects, skills, experience) lives in one file:

```
src/data/portfolio.js
```
